# Cервис для сокращения ссылок (наподобие https://bitly.com/).
Сервис предоставляет API со следующими методами:

* `POST /create?url={url}` – сохранить url и вернуть короткий идентификатор в формате JSON `{"id": "123"}`
* `GET /link/{id}` – вернуть сохраненную ссылку по идентификатору


Сервис запускается в Docker, поддерживает масштабирование и отказоустойчивость (два инстанса приложения с балансировшиком нагрузки).

### База данных

В качестве базы данных используется использовать _Redis_.

### Load Balancer

В качестве балансера используется [https://www.nginx.com/](nginx).
